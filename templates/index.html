Start

Holoeasy

Is a web Application for Computer Generated Fourier Phase-Holograms (CGHs). The application generates a matrix whose elements are a phase distribution between 0 and 2π. If this phase is placed on a Diffractive Optical Element (DOE) or a Spatial Light Modulator (SLM), it can generate a desired intensity image by an optical diffraction of Fraunhofer (Fourier). Also, the intensity distribution obtained can be simulated, by performing a numerical Fourier transform, of the hologram generated by the web application.





The steps that the user must perform to generate the hologram are:

STEP 1: Save the desired image on your PC.

STEP 2: On the main page click on the Start ítem.

The application shows a page like the following:

STEP 3: Requerimiento 1 (imagen). Click on the button Load image. A file explorer is activated and you should go to the file where you saved the image and select it.

Once the desired image has been selected, the application should display it as illustrated below. If the user verifies that it is not the desired image, he can load another one. If the image is correct, continue with the next step by clicking on the Next

STEP 4: Choose the Type of application by clicking on the radio button.

By default, Optical Application (free of speckles) is selected, for which the web application first generates a diffuser with a limited band pseudo-random phase distribution and without phase singularities. In this case the user must at least enter a plane-signal size twice as large as the hologram. Thus, the causes of speckles are avoided by the initial phase and by the finite size of the hologram. If the user does not choose to generate a hologram for an optical application, the web application first generates an initial diffuser with a random and homogeneous phase distribution. In this case, the user must enter the same size for both the signal-plane and the hologram.

Continue with the next step by clicking Next.
STEP 5: Choose to quantify the hologram or not. If you choose to quantify, enter the number of phase levels in which you want to quantify your hoogram. If you do not choose to quantify, the application generates a hologram with 255 phase levels.

Continue with the next step by clicking Next.
STEP 6: Enter values between 0 and 1 in the four parameters that allow controlling the quality of the image reconstructed by the hologram.

The regulation parameter inside window-signal serve to regulate the amplitude of the field that arrives inside the window-signal. The regulation parameter outside window-signal serve to regulate the amplitude of the field that arrives outside the window-signal. The stabilization parameter serves to improve uniformity and the relaxation parameter serves to improve the efficiency of difaction without losing uniformity. These parameters are used mainly for beam-shaping applications. Also here, the user can assign a number of iterations in each of the 10 steps of soft-coding. Different values of the parameters and the number of iterations can be assigned for the quantization stage. Click on Star to start the calculation of the hologram.


At the end of the holograms calculated by the web application are shown (unquantized and quantized). It also shows the respective reconstruction obtained and the values of the quality parameters.
The user can save the holograms in different formats
